## Installation

1.  Télécharger et installer la dernière version de [Python](https://www.python.org/downloads/)

2.  Installer Pyxel dans un terminal :

    ```
    pip install pyxel
    ```

3. Installer l'extension/bundle Python dans VS Code

4. Installer l'extension Pyxel dans VS Code

5. Lancer un programme Pyxel :    

    ```
    python -m pyxel run nom_programme.py
    ```
